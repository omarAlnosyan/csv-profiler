# CSV Profiler

Analyze CSV files in seconds.

## Quick Start

1. **Create virtual environment:**
   ```bash
   uv venv -p 3.11
   ```

2. **Activate it:**
   ```bash
   .\.venv\Scripts\activate
   ```
   (Linux/Mac: `. .venv/bin/activate`)

3. **Run CLI:**
   ```bash
   uv run python -m csv_profiler.cli data/sample.csv
   ```

4. **Run Web App:**
   ```bash
   uv run streamlit run app.py
   ```
